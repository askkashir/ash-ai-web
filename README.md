## ⚙️ How to Run

```bash
# Clone the repository
git clone https://github.com/askkashir/ash-ai-web.git

# Navigate into the project
cd ash-ai-web

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```
